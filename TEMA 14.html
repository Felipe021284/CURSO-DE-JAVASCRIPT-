Switch

En los casos donde necesitas evaluar múltiples condiciones, utilizar declaraciones if else para cada condición podría no ser la mejor solución.
La declaración switch es utilizada para ejecutar diferentes acciones basadas en diferentes condiciones.

La expresión switch es evaluada una vez.
El valor de la expresión es comparado con los valores de cada case. Si hay una coincidencia, el bloque asociado de código es ejecutado.

nota

Puedes obtener el mismo resultado con múltiples declaraciones if...else, pero la declaración switch es más efectiva en tales situaciones.

La palabra clave break

Cuando el código JavaScript llega a un break, se rompe la secuencia de un bloque switch.
Esto detendrá la ejecución de código adicional y evaluaciones de "case" dentro del bloque.

nota

Usualmente, en cada declaración de un case debería ser colocado un break.

La palabra clave default

La palabra clave default especifica el código a ejecutar si no hay ninguna coincidencia.
