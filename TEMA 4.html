Constantes

Puede haber casos en los que necesites decirle al programa que la variable no puede cambiar su valor a lo largo del programa. ¡Aquí vienen las

constantes! Las constantes son similares a las variables y se declaran con la palabra clave const:

ejemplo nota

Las constantes deben tener un valor cuando se declaran y no pueden cambiar su valor.
Las constantes son muy útiles.
Si utilizas const, le dice a cualquiera que vea tu código que esta variable nunca será asignada a un valor diferente.

Reglas de denominacion de variables

Hablemos de los nombres.

Existen varias normas para los nombres que hay que seguir en JavaScript:

los nombres de las variables deben comenzar con una letra, una barra baja o un signo de dolar $ Los nombres de las variables no contienen espacios

Los nombres de las variables sólo pueden contener letras, números, guiones bajos o signos de dólar.

Los nombres de las variables distinguen entre mayúsculas y minúsculas, lo que significa que, por ejemplo, las variables Name y name son diferentes
