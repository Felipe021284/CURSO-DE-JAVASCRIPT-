Cadenas de texto

Las cadenas de texto en JavaScript son utilizadas para guardar y manipular texto.
Una cadena de texto puede ser cualquier texto que aparezca entre comillas.
Puedes usar comillas simples o dobles.

Puedes utilizar comillas dentro de una cadena de texto, siempre y cuando no sean del mismo tipo que las comillas que rodean la cadena de texto.

Cadenas de texto

Como las cadenas de texto deben ser escritas entre comillas, hay que considerar como manejar las comillas dentro de una cadena de texto.
El carácter de escape barra inversa (\)
convierte a los caracteres especiales en caracteres de texto.
ejemplo

El carácter de escape (\) puede también ser usado para insertar otros caracteres especiales dentro de una cadena de texto.
Estos caracteres especiales pueden ser añadidos a una cadena de texto usando el signo de barra inversa.

nota

Si comienzas una cadena de texto con una comilla simple, debes terminarla también con una comilla simple.
La misma regla aplica para comillas dobles.
De lo contrario, JavaScript se confundiría.

Booleanos

En JavaScript, un Booleano (tipo de dato lógico) es aquel donde sólo puedes tener dos valores:
true (verdadero)
o false (falso).
Estos son útiles cuando necesitas un tipo de dato que sólo pueda tener uno de dos valores, tales como Sí/No, Encendido/Apagado, Verdadero/Falso.
ejemplo

nota

El valor Booleano de 0 (cero), null (nulo), undefined (indefinido), cadena de texto vacía es falso.
Todo lo que tenga un valor "real" es verdadero.
