Tipos de datos

El término "tipo de datos" se refiere a los tipos de valores con los que puede trabajar un programa. Ya nos hemos familiarizado con dos tipos de datos: texto y número.
Los textos entre comillas, como "Hello World" se llaman cadena.

Booleanos

Muy a menudo, en programación, necesitarás un tipo de datos que sólo puede tener uno de dos valores.

Por ejemplo, para crear un formulario de registro para un sitio web puede ser necesario tener una variable para comprobar si el usuario aceptó los términos y condiciones, dos escenarios posibles: sí o no.

Para ello, JavaScript cuenta con datos tipo booleanos que sólo puede tomar los valores true o false.

ejemplo nota

También hay representaciones numéricas de valores Booleanos, sabiendo que 1 significa true, y 0 es false.
