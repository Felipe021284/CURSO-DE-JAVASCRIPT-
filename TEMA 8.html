Cadenas

Hemos conocido las cadenas(strings) en lecciones anteriores,pero ahora es el momento de conocerlas mejor.

Recuerda que podemos crear una cadena introduciendo texto entre dos comillas simples o dobles.

ejemplo

Puedes usar comillas dentro de las cadenas, sólo asegúrate de que sean diferentes de los que los encierran.

Escapar caracteres

En JavaScript, la barra invertida \ es un carácter especial, llamado escape character.
Se utiliza para representar ciertas cosas en una cadena, como nuevas líneas, tabulaciones y otras cosas útiles.

Si quieres incluir una comilla en una cadena, necesitas escapar usando una barra invertida.
En este caso, no es necesario utilizar diferentes comillas de cierre.

nota

No lo confundas con el operador de división/.

Nuevas líneas

La barra invertida también se utiliza para crear nuevas líneas en el texto. Para crear una nueva línea utilizamos \n.
